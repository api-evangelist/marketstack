# Marketstack

> Real-time, intraday and historical stock market data API from APILayer. JSON REST API covering end-of-day and intraday prices across 70+ global exchanges, real-time stock prices, splits, dividends, commodities, company ratings, market indexes, bonds, ETF holdings, currencies, timezones, and SEC EDGAR filings data. Authenticate every request with the access_key query parameter; results paginate with limit (default 100, max 1000) and offset.

Generated by API Evangelist on 2026-07-22 from the artifacts in https://github.com/api-evangelist/marketstack (the provider publishes no llms.txt).

## Key endpoints

- GET /eod, /eod/latest, /eod/{date}: end-of-day bars (symbols, date_from, date_to, limit, offset)
- GET /intraday, /intraday/latest, /intraday/{date}: intraday bars (interval data, IEX US)
- GET /stockprice: real-time stock prices (Professional plan and higher)
- GET /tickers/{symbol}, /tickerslist, /tickerinfo: ticker reference data
- GET /exchanges, /exchanges/{mic}, /exchanges/{mic}/tickers: exchange reference data (ISO 10383 MICs)
- GET /splits, /dividends: corporate actions
- GET /commodities, /commoditieshistory: commodity prices (Professional+)
- GET /indexlist, /indexinfo: market indexes
- GET /bondlist, /bond, /etflist, /etfholdings: bonds and ETF holdings
- GET /cik_code, /company_name, /submissions, /company_facts: SEC EDGAR data (Business+)
- GET /currencies, /timezones: supported currencies and timezones

## Access

- [Pricing](https://marketstack.com/pricing): Free (100 req/mo) / Basic $9.99 / Professional $49.99 / Business $149.99 / Enterprise
- [Sign Up Free](https://marketstack.com/signup/free)
- [Status Page](https://status.marketstack.com)
- Errors: JSON envelope {"error": {"code", "message"}}; 401 invalid_access_key, 403 function_access_restricted (plan gating), 429 rate_limit_reached (5 req/sec) - see errors/marketstack-problem-types.yml
- Rate limits: 5 requests/second plus a monthly plan quota - see rate-limits/marketstack-rate-limits.yml
